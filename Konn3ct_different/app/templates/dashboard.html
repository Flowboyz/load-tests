Maestro Command Reference

Maestro uses clean YAML files for automation. Here are the core commands:

  • - launchApp: Launches the app defined by appId.
  • - tapOn: "Text": Clicks on an element containing the visible text or ID.
  • - inputText: "value": Enters text into the currently focused input.
  • - assertVisible: "Text": Verifies that an element is visible on the screen.
  • - back: Simulates clicking the device hardware back button.
  • - scroll: Scrolls down on the screen.
